# Arabic Medication Reminder

تطبيق Flutter عربي أولاً لتتبع كورس الدواء، مع دعم RTL، قاعدة SQLite محلية، وتنبيهات قوية شبيهة بالمنبه على أندرويد.

## التشغيل

```bash
flutter pub get
flutter run
```

## ملاحظات أندرويد

- يحتاج التطبيق إلى السماح بالإشعارات حتى تعمل التذكيرات.
- يحتاج إلى صلاحية المنبهات الدقيقة على بعض إصدارات أندرويد.
- قد تحتاج بعض الأجهزة إلى استثناء التطبيق من تحسين البطارية حتى لا تتأخر التنبيهات.
- الأصوات المدمجة موجودة في `assets/sounds` و`android/app/src/main/res/raw`.

## حدود السلامة

التطبيق لا يشخص ولا يصف ولا يقترح جرعات. هو فقط يذكر ويتتبع ما أدخله المستخدم.
